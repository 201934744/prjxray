# clb-ffsrcemux Fuzzer

## CEUSEDMUX

Configures whether clock enable (CE) is used or clock always on

| CEUSEDMUX  | CE               |
|------------|------------------|
| 0          | Always on        |
| 1          | Controlled       |


### SRUSEDMUX

Configures whether FF can be reset or simply uses D value

| SRUSEDMUX  | Resettable?      |
|------------|------------------|
| 0          | No               |
| 1          | Controlled       |
